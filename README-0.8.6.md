# HLX Migrator 0.8.6

UI cleanup release.

## Changes

- Removed the right-side details panel beside object tables.
- Object table now uses the full available width.
- Clicking **Details** opens the full diff modal directly.
- Browse-mode details also open the same modal with available source-row information.
- Existing full diff tabs remain:
  - Differences
  - Side-by-side JSON
  - Source JSON
  - Target JSON
  - Raw DeepDiff

Only the UI image needs to be rebuilt for this release.
