# HLX Migrator 0.8.3

Changes:

- Full Diff modal now includes a wide **Side-by-side JSON** tab.
- Differences table remains the default view.
- Source JSON, Target JSON and Raw DeepDiff remain available as separate tabs.
- Migration result now reports the exported definition file and file size returned by the Java ARAPI service.
- UI clarifies that Developer Studio timestamps may not change when the imported definition is identical.
- After migration the UI starts a target environment server-cache refresh so Compare can be used to verify the result.
