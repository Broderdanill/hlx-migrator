# HLX Migrator 0.9.6

UI refinements:

- Activity Log is now a true bottom drawer.
- Collapsed Activity Log no longer overlaps the workflow table rows.
- Expanding Activity Log reserves space in the main view.
- Default page size is now 100 rows for better performance.
- Page size selector still supports larger pages.
- Object type counters are compacted, e.g. 4.9k, with the full count available as tooltip.
- Header logo styling no longer adds a second visible border around the image.

Backend is unchanged from 0.9.5.
