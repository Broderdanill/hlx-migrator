# HLX Migrator 0.8.0

## Changes

- Moves large JSON comparison out of the narrow right detail panel.
- Details panel now shows only a compact summary and a differences preview.
- Adds a large Full Diff modal.
- Full Diff modal contains tabs for Differences, Source JSON, Target JSON, and Raw DeepDiff.
- Differences is the default view.
- JSON is shown full-width in a larger scrollable area.

Build and run as before with podman play kube.
